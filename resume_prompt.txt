## 项目信息

- 项目名称：万象智识库 — 企业级 RAG 多模态知识库智能问答系统
- 项目类型：RAG（检索增强生成）架构的多模态 AI 应用
- 角色：全栈开发（后端为主，前端为辅）

### 技术栈
- 后端：Python / FastAPI / LangChain / LangChain-OpenAI
- 前端：Vue 3 + Element Plus + Vite
- 知识库引擎：RAGFlow SDK（多助手动态调度）
- LLM：阿里云 DashScope 兼容接口（Qwen-Plus 文本 + Qwen-VL-Plus 视觉双模型策略）
- 多模态能力：图片理解（Vision-Language）、文档实时解析（PDF/Word/Excel/TXT）、语音输入（Web Speech API）、语音输出（SpeechSynthesis API）
- 数据库：MongoDB（会话记忆持久化）
- API 设计：REST + SSE 流式响应（支持文本/图片/文档混合输入）

### 核心工作
1. 设计并实现了完整的多模态 RAG 问答管线：意图识别 → 查询增强 → 知识库检索 → LLM 生成 → 流式输出
2. 基于 LangChain Function Call 实现双层意图分类，自动路由"知识库查询"和"一般对话"两条链路，并针对图片/文档等多模态输入场景扩展了意图判断规则
3. 集成 RAGFlow SDK，实现多知识库助手的动态发现、模糊匹配与自动调度，支持按问题主题自动选择最合适的知识库
4. 实现图片多模态理解能力：前端支持粘贴/拖拽/文件选择上传图片，后端调用 Qwen-VL-Plus 视觉模型进行图片分析，并将图片关键信息用于增强知识库检索
5. 实现文档上传实时分析：支持 PDF/Word/Excel/TXT 拖拽上传，后端基于 pypdf/python-docx/openpyxl 实时解析文档内容并注入 LLM 上下文，无需预先存入知识库
6. 实现语音交互闭环：语音输入基于浏览器原生 SpeechRecognition API 实时转文字，语音输出基于 SpeechSynthesis API 一键朗读回答，零额外 API 成本
7. 基于对话历史的查询增强模块，解决多轮对话中的代词指代和上下文依赖问题
8. 使用 MongoDB + LangChain ConversationBufferMemory 实现多会话记忆隔离与持久化
9. 设计 FastAPI SSE 流式接口，配合 Vue 3 前端实现打字机效果，支持文本/图片/文档混合请求的流式处理

### 项目亮点
- 覆盖文、图、档、音四个维度的企业级多模态知识库问答系统
- 双模型策略：文本走 qwen-plus（成本低），图片走 qwen-vl-plus（视觉强），按需切换不浪费
- 智能意图分流，多模态场景下自动判断知识库路由
- 文档即问即答，无需预先入库，实时解析注入 prompt
- 语音交互完全基于浏览器原生 API，零额外服务成本
- 全流式输出：文本/图片/文档均走 SSE 流式，打字机效果
- 向后兼容的扩展架构：图片/文档/语音均通过可选参数叠加，纯文本链路零影响
